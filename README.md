# Python & PowerShell Playground

A hybrid Python and PowerShell utility playground for system automation, event logging, and file management tasks. This project demonstrates cross-language scripting capabilities for Windows system administration.

## 📂 Project Structure
py-powershell-playground/ \
│ \
├── scripts/                    # PowerShell utility scripts \
│   ├── event-logger.ps1        # Windows event log monitoring and logging utility \
│   ├── file-ext-locator.ps1    # File extension search and location tool \
│   └── README.md               # Scripts documentation \
│ \
├── main.py                     # Python entry point and orchestrator \
└── utils.py                    # Reusable Python utility functions \


## 📝 Scripts Overview

| File | Description | Language |
|------|-------------|----------|
| `event-logger.ps1` | Windows event log monitoring and logging utility for system auditing | PowerShell |
| `file-ext-locator.ps1` | File extension search and location tool for finding files by type | PowerShell |
| `main.py` | Python entry point and orchestrator for running PowerShell scripts | Python |
| `utils.py` | Reusable Python utility functions for file operations and system tasks | Python |

## 🎯 Purpose

This playground serves as a testing ground for:
- **PowerShell automation** for Windows system administration
- **Python integration** with PowerShell scripts
- **Event logging** and system monitoring
- **File system operations** and extension-based file discovery
